# 🤖 AI Market-Research Agent (MVP)

Чат-агент для маркетингового дослідження. Telegram → розбивка задачі на підзадачі → пошук через Exa (паралельно) → зведення у звіт (Markdown).

## Структура (де що)
- **instructions.js** — УСІ інструкції/промпти агента. Тут можна читати й правити поведінку агента, не лізучи в логіку.
- **exa.js** — пошук через Exa API (один запит + паралельний по підзадачах).
- **claude.js** — виклики Claude.
- **bot.js** — головна логіка: Telegram чат, флоу research, памʼять, фідбек.

## Env-змінні (Railway → Variables)
- `BOT_TOKEN` — від @BotFather
- `ANTHROPIC_API_KEY` — Claude
- `EXA_API_KEY` — Exa (dashboard.exa.ai)

## Як працює (MVP)
1. `/start` — інструкція
2. Пишеш задачу → агент:
   - перевіряє чи не надто широка (якщо так — задає уточнення)
   - розбиває на 3-6 підзадач
   - шукає по кожній через Exa (паралельно)
   - зводить у звіт: Висновок → Знахідки → Що робити далі
3. Даєш фідбек → переробляє з урахуванням
4. `/brief <задача>` — примусово задати уточнюючі питання спершу

## Що далі (P2, по ТЗ)

- Knowledge Base на Neon (завантаження файлів + пошук/витяг)
- Режими Fast / Deep (skills)
- Памʼять у Neon (зараз у памʼяті процесу)
